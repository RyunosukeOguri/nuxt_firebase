# nuxt_firebase

* node v8.11.1
* npm 5.6.0
* firebase tool 5.0.1

## Get Started
```
$ npm install -g firebase-tools
$ cd src/
$ npm install
$ cd ../functions
$ npm install
```

## Local Server
```
$ cd src/
$ npm run dev
```

## Firebase Server

```
$ cd src/
$ npm run build
$ cd ..
$ npm run setup
$ npm run serve
```

## Deploy
```
$ cd src/
$ npm run build
$ cd ..
$ npm run setup
$ npm run deploy
```
